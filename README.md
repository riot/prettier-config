Riot default prettier config file

## Installation

Peer dependencies installation

```shell
npm in -D prettier@2
```

```shell
npm i -D @riotjs/prettier-config
```

## Usage

Set up your `.prettierrc.js` file as follows:

```js
module.exports = require('@riotjs/prettier-config')
```
